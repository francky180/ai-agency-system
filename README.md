# ClientFlow — AI-Powered Client Acquisition System

A complete, production-ready client acquisition system built with modern AI-powered tooling.

---

## What This Is

A done-for-you client acquisition package that includes:
- A premium agency website (Next.js 16, React 19, TypeScript, Tailwind CSS 4)
- Remotion-powered promotional video generation
- Client onboarding documentation (v2)
- System health monitoring dashboard
- Full technical and marketing documentation

## Who It's For

Service-based businesses, agencies, and freelancers who need a professional client acquisition system without building from scratch.

## Quick Start

```bash
cd site
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
