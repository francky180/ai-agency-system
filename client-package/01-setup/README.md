# Step 1 — Setup

## Prerequisites

Make sure you have the following installed:

- **Node.js** v18 or higher — [Download](https://nodejs.org)
- **Git** — [Download](https://git-scm.com)
- **VS Code** — [Download](https://code.visualstudio.com)
- **Claude CLI** — [Download](https://claude.ai/code)

## Verify Installation

Open your terminal and run:

```bash
node --version    # Should show v18+
npm --version     # Should show v9+
git --version     # Should show v2+
claude --version  # Should show Claude Code
```

## Project Structure

```
ai-agency-system/
├── site/               # Next.js agency website
├── client-package/     # Onboarding guides (you are here)
├── docs/               # System documentation
└── system-check/       # System verification tools
```
