# Step 3 — Run the System

## Start the Development Server

```bash
cd site
npm run dev
```

## View the Website

Open your browser and visit:

- **Homepage:** http://localhost:3000
- **System Status:** http://localhost:3000/system-status

## What You Should See

### Homepage
A premium dark-themed agency website with the following sections:
1. Hero with call-to-action
2. What This System Is
3. What You Get
4. How It Works (process)
5. System Comparison table
6. Trust / Proof section
7. Primary CTA
8. Final CTA

### System Status Page
A dashboard showing all system capabilities with live status indicators.

Once the site is running correctly, proceed to [04-verify](../04-verify/README.md).
