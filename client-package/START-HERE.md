## How to Run Locally

### Step 1 — Open the project

```bash
cd ai-agency-system
```

### Step 2 — Install dependencies

```bash
cd site
npm install
```

Wait for it to finish. You should see `found 0 vulnerabilities` at the end.

### Step 3 — Start the system

```bash
npm run dev
```

### Step 4 — Open in your browser

- **Homepage:** http://localhost:3000
- **System Status:** http://localhost:3000/system-status

You should see the full agency website with all 8 sections and the system status dashboard showing all capabilities as active.

---

## How to Verify Success

Run these two commands inside the `site` folder:

```bash
npm run lint
```

Expected: zero errors.

```bash
npm run build
```

Expected: compiles successfully with no errors.

If both pass, your system is verified and production-ready.
