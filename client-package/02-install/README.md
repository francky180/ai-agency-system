# Step 2 — Install Dependencies

## Install Website Dependencies

```bash
cd site
npm install
```

This will install:
- Next.js (App Router)
- React
- Tailwind CSS
- Motion (animations)
- TypeScript
- ESLint

## Verify Installation

After install completes, you should see:

```
added XXX packages, and audited XXX packages
found 0 vulnerabilities
```

No errors should appear. If you see vulnerability warnings, they are informational only and do not affect functionality.

Once complete, proceed to [03-run](../03-run/README.md).
